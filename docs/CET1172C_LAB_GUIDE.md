# CET 1172C - Computer Support Technician: Lab Guide
**Author:** Jarron Jackson | **Lab:** Matrix Homelab

## Lab 1 - Neb Hardware Build Documentation

### What You Will Do
Document the Nebuchadnezzar build as a professional hardware specification report - component selection rationale, installation sequence, BIOS configuration, and post-build verification checklist.

### Skills Learned
- Component identification and compatibility verification
- Hardware installation sequence
- BIOS/UEFI configuration
- POST troubleshooting
- Professional hardware documentation

### Purple Team Relevance
Understanding hardware deeply informs firmware-level attack surface. BIOS/UEFI vulnerabilities (T1542.001 - System Firmware) are an advanced persistence technique.
